API Reference

Connect to your Flask backend

Your Flask server runs at http://localhost:5000

POST /api/analyze Detect emotion from text
POST http://localhost:5000/api/analyze Content-Type: application/json { "text": "I feel really anxious today" } // Response { "emotion": "anxious", "confidence": 0.87, "scores": { "anxious": 0.87, "sad": 0.08, "neutral": 0.05 } }
POST /api/recommend Fetch Spotify tracks by emotion
POST http://localhost:5000/api/recommend Content-Type: application/json { "emotion": "happy", "limit": 5 } // Response { "tracks": [ { "name": "Happy", "artist": "Pharrell Williams", "duration": "3:53", "url": "https://open.spotify.com/track/..." } ] }
GET /api/history Fetch mood history from DB
GET http://localhost:5000/api/history?limit=20 // Response { "history": [ { "id": 1, "text": "I feel anxious...", "emotion": "anxious", "timestamp": "2026-04-11T14:22:00" } ] }
POST /api/suggestions AI mood improvement tips
POST http://localhost:5000/api/suggestions Content-Type: application/json { "emotion": "anxious" } // Response { "suggestion": "Try a 5-minute breathing exercise..." }
